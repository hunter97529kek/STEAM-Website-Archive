Welcome to the Valve Editing Resource Center!

The purpose of this site is to provide you with all the information, tools, and data you will need to make your own maps for the games Valve produces. Here you will find discussion forums, entity documentation, tutorials, and technical resources. But what to do when you can't find what you need?

Discussion Forums
The discussion forums are there for a reason - to promote and sustain discussion on all manner of editing topics. Most often, if you post a question in the discussion forums, you'll be answered quicker than if you'd emailed the support address.

IRC (Internet Relay Chat)
You can chat with other Valve ERC users at our own IRC channel on the Chatbear IRC network. The irc server is irc.chatbear.com and the channel is #valveerc. For people using the IRC client mIRC, version 5.82 or greater, you can click on this link to go directly to that channel.

Technical Support
If you are having technical problems with Worldcraft (be it setting up the editor, compiling maps, etc), the quickest course of action is to email worldcraft@valvesoftware.com (after making sure your answer can't be found in our fine resource section, of course )

When submitting questions to Worldcraft support, please include the following information.

  • What version of Worldcraft are you using?
  • Which compile tools are you using? (If you're not using Zoner's tools, just say the default ones).
  • What video card and video drivers are you using?

      Note: If you are using more than one video card (ie: in the case of a primary video card and an add-on like a Voodoo2), please list all video cards and driver versions. By driver version, I mean like, for example, the Detonator3 6.50 drivers for NVidia based cards.

  • If the problem has anything at all to do with compiling or running a map, please send the complete contents of the compile process window.
  • What operating system are you using?
  • How much memory do you have installed?
  • Please state the main problem clearly and concisely.

Getting the proper information across can really speed up the resolution of your problem.

Website related questions and feedback
Any website related mail (regarding broken links, pictures, scripts, etc) can be sent to autolycus@valvesoftware.com. I'll do my best to respond to this email, but all corrections and feedback is most certainly welcome.

Maps for Review
If you would like to submit a map for review, whether it be complete or in beta form, see this page for details.

Submittals
If you would like to submit a resource to ERC, by all means, we encourage this. There are some guidelines you should follow when preparing to submit something here.

  • All resources should be clear and concise. This doesn't necessarily mean short -- concrete/descriptive examples of things are a definite plus, but please be sure to proofread everything and check for integrity.
  • Pictures are a great teaching aid. Use them wisely - don't over-use them though.
  • Keep the number of example files to a mimimum.
  • Avoid duplication. If there's already a resource available, do something else unless you think you can do it better.
  • Leave the HTML formatting to me. A MS Word document or something similar is fine. If you want to post the resource somewhere else on the net so I can see how you'd like things formatted, that's fine too.
  • Ownership and Copyright. All submitted material remains the property and copyright of the original author. Full credit will be given for all submissions.
