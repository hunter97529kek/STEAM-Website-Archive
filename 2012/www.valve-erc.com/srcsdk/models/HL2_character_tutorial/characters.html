Creating Characters for the Source Engine

These documents give information on how to create and edit character models for the Source engine games.

  • Character Setup Overview
    Describes how to prepare and compile a character for use in the Source engine, using the SOFTIMAGE|XSI Mod Tool.
  • Making Changes to Your Character Model
    Describes how to prepare models that you've modified, before re-exporting them in Source engine formats.
  • Character Textures and Materials
    Explains how the Source engine accesses textures, and how to structure your texture directories accordingly.
  • Character Facial Animation Shapekey Set
    Lists the facial animation shapekeys required by the Source engine (with images).
  • Eye Position Setup
    Explains how to manually get and format the eye position data required by the Source engine.
  • Valve Biped Model
    Click this to open a reference image of the Valve biped rig. Place the mouse over the various parts of the rig for descriptions of its components.
