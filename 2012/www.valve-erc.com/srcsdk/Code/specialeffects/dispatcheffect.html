DispatchEffect

Provides the mechanism for dispatching a server-side request for an effect to the client-side for processing.  

The function is defined as:

void DispatchEffect( const char *pName, const CEffectData &data );

     pName
          Identifes the effect being dispatched.   This can be any name the user wishes to describe their effect (i.e. “WaterExplosion”, “GrenadeSpark”, etc).  

     data
          CEffectData instance containing data for the effect.
