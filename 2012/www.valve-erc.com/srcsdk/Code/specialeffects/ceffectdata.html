CEffectData storage class

CEffectData is a storage class that we use to transmit data (via the DispatchEffect function) to the client, where it is intercepted and acted upon.   The class is defined as:

class CEffectData
{
public:
     Vector         m_vOrigin;
     Vector         m_vStart;
     Vector         m_vNormal;
     QAngle         m_vAngles;
     int            m_fFlags;
     int            m_nEntIndex;
     float          m_flScale;
     float          m_flMagnitude;
     float          m_flRadius;
     int            m_nAttachmentIndex;
     short          m_nSurfaceProp;
     int            m_nMaterial;
     int            m_nDamageType;
     int            m_nHitBox;
     unsigned char  m_nColor;
…
}

Note: Only those variables that are changed will be transmitted to the client.
