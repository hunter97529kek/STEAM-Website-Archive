DECLARE_CLIENT_EFFECT

This macro provides a quick method for linking our effect’s identifying name to a callback function that is called on the client whenever a dispatch with the matching name is received from the server.

DECLARE_CLIENT_EFFECT( EffectName, CallbackFunction )


     EffectName
          String identifier for the effect

     CallbackFunction
          Callback function which is executed when an effect of the given name is received

          Note: Must be of type: void (CallbackFunction) ( const CEffectData &data )
