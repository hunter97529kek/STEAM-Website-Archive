Using SDK Sample Content

Sample Maps

This release contains sample maps for Counter-Strike: Source, Half-Life 2, and Half-Life 2: Deathmatch levels.

Note: Some sample maps referred to in the documentation are not yet available. These maps will be available in the following release of the Source SDK.

Counter-Strike: Source Sample Maps

The Counter-Strike sample maps are stored in a sourcesdk_content\cstrike\mapsrc\, underneath where you've installed Steam.

For example, if you've installed the Steam here, where username is your Steam login name:

C:\Program Files\Valve\Steam\SteamApps\username\

The Counter-strike sample maps would be stored in:

C:\Program Files\Valve\Steam\SteamApps\username\sourcesdk_content\cstrike\mapsrc\

To run a Counter-Strike: Source map you've built with the SDK release, see Running Compiled Maps.

Sample materials

There are sample materials in the SDK directory under sourcesdk_content\cstrike\materialsrc. To compile materials, open a command prompt and type:

"%sourcesdk%\bin\vtex" "%sourcesdk%\sourcesdk_content\cstrike\materialsrc\sample1\sample1"

Then your compiled .vmt and .vtf files will be compiled to this folder:

%sourcesdk%\..\counter-strike source\cstrike\materials

For detailed instructions on material creation, refer to Creating Materials.
