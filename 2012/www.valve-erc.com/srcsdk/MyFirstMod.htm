My First MOD

This tutorial will teach you how to create, setup and build your first simple HL2 MOD.In this MOD we will change the speed at which Half-Life 2's rocket travels through the world.

1) Create the MOD project

For the purpose of this tutorial we will call our first MOD, “MyMod”. When you make your own MOD this can be any name you wish (i.e. “QuickMod”, “TreasureTroll”, etc.)

To create a MOD project:

  1. Open the Source SDK application from Steam's Tools menu.

  2. Open Create A Mod from the Applications group.

  3. Select the Modify Half-Life 2 option.

  4. Specify where you would like the MOD's project to be placed (i.e. C:\MyMod)

  5. Enter the name of your MOD. In this example, we'll use "MyMod".

The Steam application will then copy all the relevant files to the destination directory that you specified. You now have a copy of Half-Life 2's source code at that location which you can modify.

2) Open the MOD solution

  1. Launch the Microsoft Visual C++ .NET 2003 v7.1 application.

  2. Select File->Open Solution from the menu and open the Game_SDK solution located in your MOD's src directory.

3) Alter the rocket’s speed

Open source file hl2_dll/weapon_rpg.cpp

At the top of the file, find the line:

     #define RPG_SPEED 1500

Change this line to read:

     #define RPG_SPEED 400

By altering this number, we’ve told the rocket to travel at 400 units per second (or about 33 feet per second). Previously the rocket traveled at 1500 units per second (125 feet per second). The difference in speed will be very apparent when we compile and run the MOD.

4) Set the active configuration

We'll need to set our active configuration to "Release HL2".

To do this:

  1. Select Build->Configuration Manager.

  2. Change the Active Solution Configuration drop-down list to "Release HL2"

  3. Select Close.

5) Compile the project

Now we are ready to build the solution.Choose Build->Build Solution to begin. When the requested compilation has completed there will be a server.dll and client.dll file in the MyMod/bin directory (ie: C:\MyMod\MyMod\Bin). This is the custom server and client DLL for our MOD and is one of the pieces that would be distributed to end-users.

6) Run the MOD

In the directory that you created your MOD, there will be a batch file called run_mod.bat. Use this batch file to launch your MOD. When the engine has loaded, you’ll be taken to the title screen.Hit the tilde key (~) to bring up the development console. This console will provide you with numerous tools to help you create content for the Source engine. Load a map and type “impulse 101” to give yourself a load of weapons. Select and fire the RPG to view the changes you've made.

7) You’ve built your first MOD

You’ve now setup and built your first MOD. See the SDK Documentation page for a listing of programmer documentation to help you add more features to your MOD.
