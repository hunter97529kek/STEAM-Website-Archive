Dialogs: Make Hollow

The Make Hollow command takes a selected object and generates walls out the shape, making it a hollow room.

The Make Hollow dialog will ask you to supply a value for wall thickness. Positive numbers will create the walls outward, and negative numbers will hollow inward. Each of walls that are created as separate solid objects, and are grouped together as one. You can Ungroup them to select and edit them individually. Several objects can be selected to Make Hollow at once, and the command will make separate hollow groups out of each of them.

To make an object hollow:

  1. Select an object or objects to Make Hollow.
  2. Choose Make Hollow from the Tools Menu, or press CTRL+H.
  3. Enter in a thickness value that you want the walls to be and click OK. A negative value will hollow outward.
  4. Ungroup the objects if you wish to edit the walls individually, or use the Toggle group ignore button.
