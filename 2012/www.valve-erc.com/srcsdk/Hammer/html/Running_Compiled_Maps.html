Running Compiled Maps

There is a known issue on some user's systems of Hammer not being able to run the game properly after compiling a map. If you're experiencing this problem, check the don't run the game after compiling flag in the Run Map dialog.

Follow these steps to run your compiled map after you've compiled it in Hammer:

  1. Compile your map with the Run Map feature in Hammer. Make sure don't run the game after compiling is checked.

  2. Using Windows, copy your map from the \sourcesdk directory to \maps directory of the game you're making the map for. Here are example locations for each of the core Steam games. Substitute "C:\Program Files\Valve\Steam" for the directory where you installed Steam:

    For example, for Counter-Strike: Source, if your map is located here:

    C:\Program Files\Valve\Steam\SteamApps\username\sourcesdk\cstrike_sample_content\maps

    Copy it to here:

    C:\Program Files\Valve\Steam\SteamApps\username\counter-strike source\cstrike\maps

    Where username is your Steam login name.

    For Half-Life 2 , if your map is located here:

    C:\Program Files\Valve\Steam\SteamApps\username\sourcesdk\sdkcontent\maps

    Copy it to here:

    C:\Program Files\Valve\Steam\SteamApps\username\half-life 2\hl2\maps

    For Half-Life 2: Deathmatch, if your map is located here:

    C:\Program Files\Valve\Steam\SteamApps\username\sourcesdk\hl2mp_sample_content\maps

    Copy it to here:

    C:\Program Files\Valve\Steam\SteamApps\username\half-life 2 deathmatch\hl2mp\maps

  3. Run Counter-Strike: Source, Half-Life 2, or Half-Life 2 Deathmatch via the Steam menu, depending upon which game the map has been created for.

  4. Open the Developer Console and type:

    map <mapname>

  5. Your map should load in the game.
