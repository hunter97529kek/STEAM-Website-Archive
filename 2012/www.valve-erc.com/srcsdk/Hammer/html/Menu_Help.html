The Help Menu

This menu provides easy access to a number of help sources, including several websites that will provide Valve Hammer Editor and Source Engine editing information.

Help Topics

This option brings up the Valve Hammer Editor help file.

Entity Guide

This option will bring up the Entity Guide which contains in depth information on all of the entities, including appropriate entity variables, spawnflags, and usage information.

Valve Hammer Editor homepage

This will take you to the home page for the Valve Hammer Editor.

About…

This brings up the About dialog which contains version and copyright information.
