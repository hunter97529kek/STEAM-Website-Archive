Valve Hammer Editor User's Guide

Ready to jump into the exciting world of level editing? You've come to the right place. This User's Guide is designed to familiarize you with the Hammer Editor basics. It will introduce you to the essential elements that go into every level, and help you understand how all the pieces fit together.

Getting Started

  • Introduction to Editing - A introduction to the basic principles of editing with the Hammer Editor.
  • Your First Half-Life 2: Deatmatch Room - A step-by-step guide to making a basic level.
  • 3D and 2D Views - Features of the 3D and 2D viewports.
  • Creating Brushes - How to create solid shapes.
  • Applying Materials - How to texture objects.
  • Reshaping Solids - How to modify existing solid geometry.
  • Creating Entities - How to create entities for gameplay.

Hammer Reference

  • Menus - Reference guide to the menus and their contents.
  • Dialogs - Reference guide to various dialog boxes.
  • Toolbars - Description of the toolbars and their icons.
  • Hotkey Reference - Listing of all keyboard shortcuts.
  • Entity Guide - Finding information on specific entities.

Other Level Editing Topics

  • Grouping and VisGrouping - How to make groups for organization and selection.
  • Creating and Using Prefabs - How to use prefab objects.
  • Adding Water - Adding water volume brushes to maps.
  • Leaks Explained - Description of leaks, and how they can be fixed.
  • Displacement Surfaces - Creating displacement geometry (terrain).
  • Cubemaps - Adding cubic environment maps to levels.
  • Entity I/O - How to hook up entity inputs and outputs.
  • Ladders - Adding ladder entities to levels.
  • Overview of Prop Types - Description of the various model prop entities.
  • Physics Entity Overview - How to use the various entities that affect the physics system.
  • Cables - How to add cables and ropes to levels.
  • Making 3D Skyboxes - How to create a 3D geometry skybox.
  • Making Menu Background Maps - Creating the 3D menu backgrounds.

  • Making levels for Counter-Strike: Source - A description of the entities needed for CS:S gameplay.
