Using Outputs and Inputs

Outputs are used to control game events by adding outputs to entities that are triggered to activate other entities.

Outputs

Lists all of the Outputs for the current entity, and lets you add additional Outputs that the selected entity fires.

How to add an output to an entity:

  1. Select the entity you wish to add the output to.
  2. Choose Properties from the Edit Menu to open up the Object Properties dialog for the entity, and switch to the Outputs tab.
  3. Click Add... to add an output to the entity.
  4. Change the parameters on the output to fire an input on another entity.

Inputs

Shows all of the Inputs to the current entity. Double-click an Input to go to the Outputs tab of the entity that originated the Input. Double-click an Input to switch to the entity that contains the connected output.
