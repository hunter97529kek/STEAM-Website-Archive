Toolbars: Map View

2D Grid

Toggles the 2D grid -- gridlines that show up in the 2D Views. See Show Grid, and Grid Settings for more information.

3D Grid

Toggles the 3D grid -- gridlines that show up on all brush solids in the 3D View.

Smaller Grid, Larger Grid

These decrease and increase the grid size. See Show Grid, and Grid Settings for more information.

Load and Save Window State

These buttons are used to save and restore window configurations when the Use independent window configurations option is being used.
