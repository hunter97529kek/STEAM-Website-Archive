Entity Definitions (.FGD) File Updates

These changes to the .FGD files have been made from the initial SDK release. The .FGD files are used by the Valve Hammer editor to provide a list of usable entities for each game. These files are automatically received when you get a Source SDK update.

March 9, 2005 Update · Initial clean up pass of all .FGD files.
