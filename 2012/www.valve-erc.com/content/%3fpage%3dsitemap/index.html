Site Map
This section provides an overview of the website.

news

    This is the main page of the site. On it you will find news updates and editorial content. If you would like to submit an item of news, please contact me. Valid news topics would be new editing web sites, new editing utilities, mapping contests, etc.

knowledge base

    The knowledge base should be the first place you go if you've got a question or problem you're trying to solve. Enter a keyword or phrase and search, or press the Display All button to display all the articles in the knowledge base.

resources

    This section contains all manner of resources related to the Worldcraft editor and editing in general. This includes setup guides, editing tricks and tips, in-depth editing tutorials, and technical reference information. If you'd like to submit a resource, please review the submital criteria in the contact section.

    setup guide
    This is an all-purpose setup guide. Fill in the appropriate information and you'll be given customized Worldcraft setup information.

    FGDdb
    This page contains links to a large number of game data files.

entities

    The entities section contains entity documentation and game data files (proprietary Worldcraft entity data files) for all of Valve's games, and some of the popular mods as well. If you are interested in submitting entity information for a mod, please review the submital criteria in the contact section.

utilities

    Here you will find programs that are invaluable to the Half-Life map developer. Some of these programs are required for the purpose of map editing, and others are here to make your life easier. Collect them all for the ultimate map design toolkit. (Caution, some assembly required.)

files

    This section links to the VERC files section at 3ddownloads.com. All of the various files on this site are stored there.

contact

    If you need to contact someone for help or would like to offer feedback or submit something to the site, this is the place to go. You'll find the various support email addresses and web sites, as well as criteria for anything you might want to submit to this site. Giving the proper information, whether you looking for help or looking to offer help, will help speed things along.

forums

    Our forums are powered by the almighty Chatbear messageboard system. More than just a messageboard, it's a way of life. Use these forums to discuss editing a specific game, to get technical help with Worldcraft, offer feedback on the site, or just to blow off a bit of steam (be nice though!).
