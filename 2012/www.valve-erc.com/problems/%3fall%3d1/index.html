5. Why do icon sprites look fine, but other sprites have a black box around them?
For entities that have a sprite associated with them, you need to set the Render Mode to Additive to get rid of the black box. Also be sure to set the FX Amount to a value between 0 and 255 - the higher the value, the more solid the sprite will appear.
