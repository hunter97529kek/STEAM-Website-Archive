Half-Life SDK

The current version of the Half-Life SDK is 2.2, which includes new functionality for mod authors, greatly increasing their power to create a wide variety of games using the Half-Life engine. The new features included with version 2.2 are the inclusion of voice chat, the new spectator technoogy, and the source code for Deathmatch Classic.

The SDK is available for download from the Valve Editing Resource Center
