Joining a Discussion List

Server Operators and MOD programmers are an important part of Half-Life's on-line community. To help them help each other we have set up some mailing lists that provide a forum for server administrators and MOD programmers to discuss issues that pertain to running a Half-Life dedicated server or creating a Half-Life MOD. We have also created a mailing list for Half-Life release announcements, which is where you will hear first about upcoming Half-Life releases and updates.

to recieve mail about upcoming Half-Life releases:
click here http://list.valvesoftware.com/mailman/listinfo/hlds_announce
archive exists here http://list.valvesoftware.com/mailman/private/hlds_announce/

for discussion of windows server issues:
click here http://list.valvesoftware.com/mailman/listinfo/hlds
archive exists here http://list.valvesoftware.com/mailman/private/hlds/

for discussion of linux server issues:
click here http://list.valvesoftware.com/mailman/listinfo/hlds_linux
archive exists here http://list.valvesoftware.com/mailman/private/hlds_linux/

for discussion of Half-Life MOD programming issues:
click here http://list.valvesoftware.com/mailman/listinfo/hlcoders
archive exists here http://list.valvesoftware.com/mailman/private/hlcoders/
