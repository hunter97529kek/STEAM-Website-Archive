Step By Step :: Steam And Your Computer

How much space will Steam take up on my hard drive if I accept the default cache size?
In general it depends on the application, however the default is selected for optimum application/game performance. In the case of Half-Life products, the default is 512MB.


How do I remove the beta software from my computer?
Open the Account window in the Valve interface and click “Delete Account”. You will be asked to enter your password and if you are sure you want to delete the account. Click “OK” and the application will be moved from your PC.


Where are the files stored on my PC?
As part of the start-up you will be asked to create a Steam directory. This directory can be placed anywhere on you local hard drive. The default location is C/:Steam/. All information is stored in this directory.

Does the Steam Application install over or modify any of my current Counter-Strike files, including the client.dll or anything resulting in a modification of counter-strikes core files?
No. Running steam will not disturb or ruin existing installations of the games.

Will my computer run a Valve game?
These are the minimum system requirements to run the games:

- Pentium 200 MHz with 32MB RAM
- Microsoft Windows 98/ME/2000/NT 4.0 or 2000
- SVGA, high-color (16 bit) video card
- 2G HD space
- Win-compatible soundcard
- Mouse, keyboard
- 32-bit Internet service with 384K+; DSL, Cable Modem, T1 or higher

Recommended system:

- Pentium II 200+ MHz, 128 MB RAM
- 3D accelerator card (OpenGL or Direct 3D)
- 32-bit Internet service with 512K+

Does this software work with a DHCP connection?
Yes. The technology works with DHCP connections. Our electronic ticket system accounts for changes in your assigned IP address. If you IP address changes between sessions you will be asked to enter your password.

What is the minimum Internet connection speed required?
The term “broadband” refers to any Internet connection capable of sustained throughput of at least 256Kbps. We have designed it to deliver the optimum experience at 384Kbps. The technology will work at lower speeds but the experience will be slower and may require longer waiting periods before content is available. The only forms of phone line transmission omitted by this restriction is ISDN (128Kbps) and traditional dial-up modems (<56Kbps).

Can I put my cache folder anywhere?
Yes. After you extract and open the Steam.exe, you have the option of creating a cache folder in the desired location.

How to I move/rename a Steam cache directory?
1. In the Steam "Configure" page, UN-check "Accept default cache size and location on subscribe"
2. Close Steam (and the game if it is running)
3. Move/rename the cache directory
4. Launch Steam, then launch the corresponding game
4a. Steam will ask you for the location of the cache directory, point it to the new one
5. If you want to accept the default cache size, re-check "Accept default cache size and location on subscribe"
